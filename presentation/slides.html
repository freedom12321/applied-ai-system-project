05 — What I Learned

AI collaboration, both directions

Helpful

When I switched providers, Claude verified the real Gemini SDK source instead of trusting a summarized doc page — which described an API shape that turned out not to match the actual package.

Flawed

Claude's own wording — "strict," specific to the old provider — leaked into three files describing Gemini's different mechanism, and sat there until I asked for a documentation review pass.

Limitation, stated plainly

The confidence score measures how much correction was needed — not whether the result is actually right. A profile needing zero corrections can still misread what someone meant.
